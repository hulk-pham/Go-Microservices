# Golang project

## Feature
- Gin
- JWT
- GORM
- Viper
- Swagger
- Upload S3
- Unit test


## Start

1. Create a .env file at the root of the project and add the following:
   ```
   cp .env.example .env
   ```
2. Run `go mod download`.
4. Run `make run` or `make dev`.
